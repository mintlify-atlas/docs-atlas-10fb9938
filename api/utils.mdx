## Overview

The Utilities module provides reusable helper functions for common operations throughout the AgroVets application. These functions handle formatting, data transformation, and other shared logic.

## Import

```typescript
import { formatDate } from '@/utils';
```

## Functions

### formatDate

Formats a JavaScript Date object into an ISO 8601 date string (YYYY-MM-DD format).

<ParamField path="formatDate" type="function">
  Converts a Date object to ISO date string format
</ParamField>

#### Signature

```typescript
export function formatDate(d: Date): string
```

#### Parameters

<ParamField path="d" type="Date" required>
  The Date object to format
</ParamField>

#### Return Value

<ResponseField name="return" type="string">
  ISO 8601 date string in YYYY-MM-DD format (e.g., "2026-03-04")
</ResponseField>

#### Usage

```typescript
import { formatDate } from '@/utils';

const today = new Date();
const formatted = formatDate(today);

console.log(formatted); // "2026-03-04"
```

#### Real-World Examples

```typescript
// Display formatted date in UI
function AppointmentCard({ date }: { date: Date }) {
  return (
    <View>
      <Text>Appointment Date: {formatDate(date)}</Text>
    </View>
  );
}

// Format dates for API requests
async function fetchRecords() {
  const startDate = formatDate(new Date('2026-01-01'));
  const endDate = formatDate(new Date());
  
  const response = await fetch(
    `https://api.example.com/records?start=${startDate}&end=${endDate}`
  );
  
  return await response.json();
}

// Create date ranges
function getDateRange() {
  const today = new Date();
  const yesterday = new Date(today);
  yesterday.setDate(yesterday.getDate() - 1);
  
  return {
    start: formatDate(yesterday),
    end: formatDate(today),
  };
}
```

#### Edge Cases

```typescript
// Current date
formatDate(new Date()); // "2026-03-04"

// Specific date
formatDate(new Date('2026-12-31')); // "2026-12-31"

// Past date
formatDate(new Date('2020-01-01')); // "2020-01-01"

// Date with time (time is ignored)
formatDate(new Date('2026-03-04T15:30:00')); // "2026-03-04"
```

## Implementation Details

The `formatDate` function uses the native `toISOString()` method and splits on the 'T' separator to extract only the date portion:

```typescript
export function formatDate(d: Date) {
  return d.toISOString().split('T')[0];
}
```

<Note>
  This function always returns dates in UTC timezone. If you need locale-specific formatting or timezone handling, consider using libraries like `date-fns` or `moment.js`.
</Note>

## Extending Utilities

To add more utility functions, export them from the same module:

```typescript
// Format currency
export function formatCurrency(amount: number): string {
  return `$${amount.toFixed(2)}`;
}

// Capitalize string
export function capitalize(str: string): string {
  return str.charAt(0).toUpperCase() + str.slice(1);
}

// Truncate text
export function truncate(text: string, maxLength: number): string {
  if (text.length <= maxLength) return text;
  return text.slice(0, maxLength) + '...';
}
```

## Best Practices

- Keep utility functions pure and side-effect free
- Add TypeScript type annotations for better IDE support
- Write unit tests for all utility functions
- Document expected input and output formats
- Handle edge cases gracefully
