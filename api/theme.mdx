## Overview

The theme system provides centralized design tokens for maintaining consistent colors and spacing throughout the AgroVets application. All values are exported from a single module for easy access and modification.

## Import

```typescript
import { colors, spacing } from '@/theme';
```

## Colors

The `colors` object contains the primary color palette used across the application.

<ResponseField name="colors" type="object">
  Color palette object with theme color definitions
  
  <ResponseField name="primary" type="string" default="#0a84ff">
    Primary brand color - vibrant blue used for primary actions and highlights
  </ResponseField>
  
  <ResponseField name="background" type="string" default="#ffffff">
    Default background color - white for main content areas
  </ResponseField>
  
  <ResponseField name="text" type="string" default="#222">
    Primary text color - dark gray for readable body text
  </ResponseField>
</ResponseField>

### Usage

```typescript
import { colors } from '@/theme';

const styles = StyleSheet.create({
  container: {
    backgroundColor: colors.background,
  },
  title: {
    color: colors.text,
  },
  button: {
    backgroundColor: colors.primary,
  },
});
```

## Spacing

The `spacing` object provides consistent spacing values using a scale-based system.

<ResponseField name="spacing" type="object">
  Spacing scale object with numeric pixel values
  
  <ResponseField name="s" type="number" default="8">
    Small spacing - 8px for tight layouts and compact elements
  </ResponseField>
  
  <ResponseField name="m" type="number" default="16">
    Medium spacing - 16px for standard margins and padding
  </ResponseField>
  
  <ResponseField name="l" type="number" default="24">
    Large spacing - 24px for generous whitespace and section separation
  </ResponseField>
</ResponseField>

### Usage

```typescript
import { spacing } from '@/theme';

const styles = StyleSheet.create({
  container: {
    padding: spacing.m,
    marginBottom: spacing.l,
  },
  item: {
    marginTop: spacing.s,
  },
});
```

## Complete Export

```typescript
export const colors = {
  primary: '#0a84ff',
  background: '#ffffff',
  text: '#222',
};

export const spacing = {
  s: 8,
  m: 16,
  l: 24,
};
```

<Note>
  These design tokens are defined in `src/theme/index.ts` and can be extended with additional colors, spacing values, or other design tokens as your application grows.
</Note>

## Best Practices

- Always use theme values instead of hardcoded colors and spacing
- Stick to the spacing scale (s, m, l) for consistent visual rhythm
- Use `colors.primary` for interactive elements like buttons and links
- Use `colors.text` for body text to ensure readability
