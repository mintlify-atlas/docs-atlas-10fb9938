# Welcome to AgroVets App

AgroVets App is a mobile application template built with React Native and Expo, designed to provide a solid foundation for agricultural veterinary services applications. It features a modern navigation system and a well-organized project structure.

## What is AgroVets App?

AgroVets App is a small template/seed project for building React Native applications using Expo. It provides a pre-configured navigation system with bottom tabs, stack navigation, and drawer navigation patterns, allowing you to quickly start building features without setting up the basic infrastructure.

<Note>
This is a template project designed to be customized for your specific agricultural veterinary service needs.
</Note>

## Technology Stack

The app is built with modern React Native technologies:

- **React Native** `0.81.5` - Cross-platform mobile framework
- **React** `19.1.0` - UI library
- **Expo** `~54.0.25` - Development framework and tools
- **React Navigation** `7.x` - Navigation library
  - Native Stack Navigator
  - Bottom Tabs Navigator
  - Drawer Navigator (prepared)
- **TypeScript** `~5.9.2` - Type safety and better DX

## Who is This For?

AgroVets App is ideal for:

- **Developers** building agricultural or veterinary service applications
- **Teams** looking for a pre-configured React Native + Expo starter template
- **Projects** that need a solid navigation foundation from day one
- **Anyone** wanting to quickly prototype mobile app ideas with modern tools

<Tip>
The template includes three sample screens (Home, Profile, Settings) connected through bottom tab navigation - perfect for understanding the navigation flow before building your own features.
</Tip>
